## Skills Acquired

### SQL (Data Extraction & Transformation)
- Writing complex queries using:
  - `CTEs (WITH clauses)`
  - `Window Functions` (`ROW_NUMBER`, `RANK`, `LAG`, `LEAD`)
  - `JOINS` (INNER, LEFT, RIGHT)
  - `Subqueries`
- Data cleaning and preparation
- Aggregations and grouping
- Deriving insights from raw datasets
- Solving real-world SQL problems (DataLemur-style case questions)

---

### Excel (Analysis & Data Preparation)
- Data cleaning and preprocessing
- Pivot Tables & Pivot Charts
  - `XLOOKUP`
  - `IF`, `IFS`
  - Conditional formatting
- Exploratory analysis
- Building analytical dashboards
- Preparing structured datasets for visualization tools

---

### Tableau (Visualization & Storytelling)
- Connecting and preparing data sources
- Data modeling:
  - Joins, unions, relationships
  - Logical vs physical layers
- Metadata management:
  - Dimensions vs measures
  - Discrete vs continuous fields
- Calculations:
  - Calculated fields
  - **LOD expressions** (`FIXED`, `INCLUDE`, `EXCLUDE`)
  - Date, string, logical, and aggregate functions
- Parameters and parameter-driven analysis
- Actions:
  - Filter actions
  - Highlight actions
  - Dashboard interactions
- Dashboard design best practices:
  - Layout and formatting (Containers)
  - Usability and storytelling
